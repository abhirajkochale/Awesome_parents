# AwesomeKids Design System

## 1. Overview & Tone
**Brand Type:** Preschool / Kids education platform
**Tone:** Fun but structured, playful but not messy, modern, and kid-friendly.
**Core Layout Style:** Bento UI System (grid-based, colorful modular blocks, each section acts as a card/block with dashboard-like clarity).

## 2. Color System
Extracted directly from the AwesomeKids logo blocks:
- **Backgrounds:** Light neutral (White #FFFFFF or soft grey #F8F9FA) to maintain a bright, clean, breathable canvas. Avoid dark-heavy UI.
- **Pink:** `#E91E63` - Used for primary accents / highlights.
- **Blue:** `#2196F3` - Used for primary navigation / structured data blocks.
- **Red:** `#F44336` - Used for loud call-to-actions / urgent elements.
- **Green:** `#4CAF50` - Used for success states / curriculum modules.
- **Orange:** `#FF9800` - Used for secondary buttons / playful badges.
- **Purple:** `#9C27B0` - Used for creative sections / testimonials.

## 3. Typography
**Style:** Rounded, friendly, large, highly readable.
**Characteristics:**
- **XXL Heading:** 3.5rem (56px) - Extra Bold
- **XL Heading:** 2.5rem (40px) - Bold
- **Large Heading:** 2rem (32px) - SemiBold
- **Body Text:** 1.125rem (18px) - Regular / Medium

## 4. Layout & Spacing
- **Spacing:** Airy, breathable. Gutters between Bento blocks should be generous (e.g., 24px or 32px). Sligthly playful spacing.
- **Border Radius:** Heavily rounded (`border-radius: 24px` for large cards/blocks, `16px` for nested elements).
- **Shadows:** Soft, diffused shadows (`box-shadow: 0 8px 32px rgba(0,0,0,0.06)`) to lift cards off the neutral background cleanly without harsh contrast.

## 5. Components
### Buttons
- **Shape:** Color-coded (based on palette), rounded (pill-shaped), and large.
### Cards (Bento Blocks)
- **Shape:** Different colors per section, soft shadows, rounded corners forming a strict but colorful grid.
### Inputs
- **Shape:** Clean, highly rounded borders with bright focus states.
### Navigation
- **Style:** Simple, playful, sticky top-bar with pill-shaped elements and color-coded links.
